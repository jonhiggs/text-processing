NAME

demarkdown — print markdown without formatting

SYNOPSIS

demarkdown [FILE]...

DESCRIPTION

The demarkdown utility removes all formatting from markdown. This is helpful when performing text-processing in a pipeline.

When content is stripped from the document, the position of the remaining content is preserved.

BUGS

demarkdown does not handle tables. It also doesn’t handle parentheses in links. GNU 2024-01-20 demarkdown(1)
