NAME

unfold — combine lines, reversing the affects of fold(1)

SYNOPSIS

unfold [FILE]...

DESCRIPTION

The unfold utility joins lines to create single-line paragraphs. Input is read from the supplied files or standard in if no files were provided.

Leading and trailing whitespace is removed when it interrupts the flow of a paragraph. Duplicate empty lines are also removed.

Line breaks are preserved when they they add structure to the content, e.g. when separating list elements.
