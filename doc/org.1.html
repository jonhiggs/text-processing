NAME

org — print the first and last sentence from each paragraph fold(1)

SYNOPSIS

org [FILE]...

DESCRIPTION

The org prints each paragraphs first and last sentences, showing how ideas flow from one paragraph to the next.

Lines can be either hard or soft wrapped, but paragraphs must be separated with an empty line.

If no FILES are supplied, input is read from standard in.

HISTORY

This tool was inspired by the line: “org. which prints the first and last sentence of each paragraph, so you can analyze paragraph transitions and the flow of ideas within your writing.” (UNIX Text Processing, by Dale Dougherty and Tim O’Reilly)

It originated under the same name in Writers WorkBench (wwb) by Lorinda Cherry and Nina Macdonald of Bell Labs. This isn’t a clone since I’ve never used or even seen the original.
